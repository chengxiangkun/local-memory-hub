# 低保真原型：Local Memory Hub V1

## 1. 目标

本文件用文字 wireframe 描述 V1 关键页面的低保真原型。

它不定义最终视觉风格，只定义：

- 页面布局
- 信息优先级
- 核心操作
- 状态展示
- 详情面板
- 失败和空状态

## 12. 关键交互约束

### 12.1 快捷导入不能绕过导入管线

所有快捷入口最终都必须显示在 Inbox 和源资料库。

### 12.2 解析成功不等于入记忆成功

UI 必须分开显示这两个状态。

### 12.3 删除必须说明影响范围

删除源文件前，用户必须看到是否会影响：

- 原始文件
- 解析文本
- 文本片段
- 向量索引
- 图谱节点
- 图谱关系
- 历史问答引用

### 12.4 无模型时仍然可用

搜索/问答页不能因为没有模型配置而空白。必须展示搜索兜底结果。

### 12.5 图谱不是摆设

图谱必须支持：

- 搜索
- 聚焦
- 详情
- 追溯
- 污染过滤
