# V1 实现状态清单

### 2. 资料变成可用记忆

已完成：

- 文本和 Markdown 解析。
- PDF/图片/媒体解析失败时中文错误。
- 媒体元数据读取和音频抽取。

已完成：

仍有限制：

### 5. 图谱探索

已完成：

- 图谱 API。
- 图谱搜索。
- SVG Obsidian 风格原型。
- Force Graph Spike。
- 污染节点默认排除。
- 根据已有文本片段重建图谱索引。
- 根据已有文本片段重建向量索引。
- 跨资料共享关键词节点，支持不同笔记通过相同关键词产生关联。
- 图谱节点和关系分批加载动效。
- 默认图谱限制加载数量，避免一次性全量渲染。
- 节点点击后懒加载一跳邻居，详情面板不刷新整页。
- 图谱搜索返回命中节点子图，而不是只在全量图里高亮。
- 社区概览按节点类型聚合，并支持点击社区节点下钻到类型子图。

仍有限制：

- 正式 Force Graph 接入未替换当前 SVG 原型。
- 截图导出未完整产品化。
- 高级鼠标互动已后置。

## 当前最重要的工程限制

- Tauri 桌面壳因本机缺 Rust/Cargo 暂未验证。
- SQLite 当前仍通过系统 `sqlite3` CLI，正式版需替换为内嵌 SQLite。
- API Key 当前是本地明文文件 + `600` 权限隔离，正式版需接系统密钥存储或加密文件。
- 本机能力检查已可提示缺少的解析和打包依赖。

## 下一步最短路径

1. 正式化存储层：替换系统 `sqlite3` CLI。
2. 正式化桌面壳：安装 Rust 后继续 Tauri Spike。
3. 正式化图谱渲染：用 Force Graph 替换 SVG 原型。
4. 做飞书/腾讯文档真实 OAuth/API 拉取。
5. 做截图导入和 OCR 本地解析。
