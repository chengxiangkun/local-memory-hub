# Local Memory Hub 用户使用说明

## 第一次启动

1. 启动本地 API 和 Web 页面。
2. 打开设置中心，确认数据保存位置。
3. 在“本机能力检查”里确认 SQLite、FFmpeg、PDF/OCR 等能力状态。
4. 如果要使用大模型问答，在“模型 Provider”里配置 API Key 和模型名称。

## 导入资料

支持入口：

- 文本：粘贴内容后导入。
- 文件：选择本地文件后保存为源资料。
- 链接：保存分享链接或网页地址。
- 外部文档：第一版保留飞书文档和腾讯文档连接入口。

导入后会发生：

- 源资料保存到本地数据目录。
- 进入解析流程。
- 生成文本片段、向量索引和图谱节点。
- 源资料库显示处理状态、是否入记忆、是否可追溯。

## 源资料库

源资料库用于查看和管理原始资料。

- 左侧是文件夹结构。
- 可以新建本地文件夹。
- 可以在表格的“文件夹”列把资料移动到指定文件夹。
- 飞书和腾讯文档资料会保留平台来源，后续真实同步会写入同一套结构。

## 搜索与问答

- 没有配置大模型时，系统使用本地搜索和向量/图谱上下文兜底。
- 配置大模型后，可以选择问答模型，生成带引用的回答。
- 每次问答默认会沉淀为记忆，可在问答页关闭。

## 污染治理

- 可以从图谱节点或问答引用标记污染。
- 被隔离的内容默认不参与普通搜索、问答和图谱展示。
- 删除源资料时可以选择是否同时删除派生的文本片段、向量和图谱。

## 数据和升级

数据保存在设置中心展示的本地目录中。升级程序不应覆盖该目录。更换目录必须经过备份、复制、校验和重启。
